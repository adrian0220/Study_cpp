# 6. 상속

## 1) 상속(Inheritance)이란?

<br/>

### 1-1) 개요

<br/>

상속(inheritance)은 추상화, 캡슐화와 더불어 객체 지향 프로그래밍을 구성하는 중요한 특징이다. <br/>
상속의 가장 큰 장점은 코드 재활용성이다. <br/>
클래스 간의 계층적 관계를 구성하게하여, 앞서 배운 다형성의 토대를 마련한다. <br/>

<br/>
<br/>

### 1-2) 클래스 상속

<br/>

클래스 상속의 개념을 이해하기 위해 앞서 사용했던 Car의 개념을 확장해보자. <br/>

# 1. C++ 란?

### - C++ 란?  
    
AT&T 벨 연구소의 비야네 스트롭스트룹이 C언어 기반으로 1983년 발표하여 발전한 프로그래밍 언어
   
   
   
### - C++ 의 특징
    
* 기존 C의 절차 지향적 언어 특성에 객체 지향적 언어의 특징을 추가
* 템플릿으로 대변되는 일반화 프로그래밍 방식 추구
* C 표준 라이브러리 그대로 사용 가능
   
   
   
### - C++의 기능
   
C++ 컴파일로는 아래의 기능을 제공한다.   
   
* 다중 상속
* 템플릿
* 연산자 오버로드
* 예외 처리
* 실행시 형 식별
